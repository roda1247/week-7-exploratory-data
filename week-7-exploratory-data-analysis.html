The n() Function

The n() function does not need any input. It simply counts the number of rows in the data frame or tibble.

babynames |> 
  summarize(row_n = n())
# A tibble: 1 × 1
   row_n
   <int>
1 144000

The n_distinct() Function

babynames |> 
  summarize(row_n = n(),
            name_n = n_distinct(name))
# A tibble: 1 × 2
   row_n name_n
   <int>  <int>
1 144000   3373

dplyr - group_by()

NOTE The group_by() function is never used alone. It is always used before either summarize() or mutate() to communicate to those functions how to perform thier calculations.

pollution |> 
  summarize(mean = mean(amount),
            sum = sum(amount),
            n = n())
# A tibble: 1 × 3
   mean   sum     n
  <dbl> <dbl> <int>
1    42   252     6
pollution |> 
  group_by(size) |> #only new line of code
  summarize(mean = mean(amount),
            sum = sum(amount),
            n = n())
# A tibble: 2 × 4
  size   mean   sum     n
  <chr> <dbl> <dbl> <int>
1 large  55.3   166     3
2 small  28.7    86     3
babynames |> 
  group_by(sex) |> 
  summarize(total = sum(n))
# A tibble: 2 × 2
  sex       total
  <chr>     <dbl>
1 F     147043618
2 M     163865724

  • Do groupwise operations with group_by()

  • Connect operations with %>% or |>
